Computer Science Classes

Inroduction to Computer Science

A year long class split into two semesters. The first course focused on problem solving using computers and introduced the topics: problem decomposition, development of graphical user interfaces, development of simple algorithms, and object-oriented techniques. The second course also placed emphasis on static and dynamic variable structures which were used to implement well-known algorithms such as: searching, sorting, list, and string processing. Both of these courses were taught in Java.

Data Structures and Algorithms

This course covered fundamental data structures which include: lists, stacks, queues, binary search trees, AVL trees, hash tables, and heaps. This course also included algorithm analysis which algorithms such as: insetion sort, mergesort, heapsort, quicksort, and searching algorithms.

Occidental College

Computer Science Classes

Computer Organization

This class covered the basics of the inner workings of computers and focused especially on the hardware-software interface. The class emphasized that lower-level hardware should be kept in mind while designing effiecent programs. Important topics included: memory hierarchy, parallel computing, and principles of digital logic. The course also introduced smaller programs using assembly language and had an introduction to the C programming language.

Programming Languages

This course focused on the inner makings of a programming language. Things that were discussed during this class included parsing and interpreting a language. An example of components that were introduced were optional vs mandatory spaces, parsing through mathematical statements, and the making of your own syntax. The final project could have been coded in either Java or Python.

Machine Learning

This course focused on the fundamentals of machine learning without relying on the many built-ins avaible for languages like Python. This course covered many mathematical preliminaries for machine learning, supervised learning, unsupervised learning, and deep learning. The course emphasised that many machine learning programs could be broken down into two categories: defining a cost function with proper parameters that can accurately describe the cost and optimizing the cost function. This class was focused on coding in Python.

Robotics

This course focused on creating and exploring various subfields of robotics. Subfields that were explored include: hardware which includes sensor, actuators, and circuits, navigation and SLAM, planning, learning, and social/cognitive robotics and human-robots interaction. This class emphasised working with real hardware and exploring the diffuclties of have both software and hardware work optimally together.

Databases

This course focused on exploring specialized algorithms that can help ensure consistency and efficient access even with very large datasets. This class covered relational databases, mySQL, er diagrams, and newer graph databases. During this course, we worked with real world data to learn how to use and manipulate data in a mySQL database.
